# 绿杉树项目介绍

这是一个名为"绿杉树"的桌面应用程序，它是一个针对"红杉树英语学习平台"（hssenglish.com）的第三方客户端工具。以下是该项目的详细介绍：

**项目概述：**
- **项目名称**: 绿杉树
- **类型**: 第三方客户端工具，针对红杉树英语学习平台
- **技术栈**: Python、PySide6、mitmproxy
- **打包方式**: 使用EXE可执行文件分发，支持一键安装和卸载

**特性如下：**
1. _个性化_ ：可替换各类URL实现个性化
  实例：在设置界面的URL替换规则中设置
  `{
    "https://style.hssenglish.com/v9.0/student/images/theme-lantern/common/bg-1080.png": "https://i1.hdslb.com/bfs/article/4b4d62a69f10b55163fb3ee4da1d2ca4fa7231e8.png"
}`
  即可将“千灯会”主题的背景替换为音理的图片
2. _自动登录_ ：可保存账号密码并在启动时自动登录
3. _破解_ ：开启绿杉树模式后，答案全部变成“a”

**注意事项：**
本程序仅供编程学习使用，请在学习后自行删除。因其本质上是一个中间人攻击(MITM)工具，用于修改在线教育平台的通信数据，可能存在违反服务条款的风险，请谨慎使用。

**技术架构：**
1. GUI界面
  使用PySide6构建现代化图形界面
  内嵌浏览器组件显示红杉树平台内容
  提供全屏模式和无边框窗口设计
  绿色边框表示启用模式，红色表示禁用模式
2. 代理系统
  使用mitmproxy框架实现HTTP(S)流量拦截
  在后台线程中运行代理服务
  动态分配端口并保存到配置文件
3. 配置管理
  支持用户名密码配置
  支持自定义URL替换规则（JSON格式）
  支持自定义Python代码函数进行高级操作

**项目特色：**
1. 自动化部署
  - 通过Windows任务计划程序实现开机自启
  - 支持管理员权限运行
  - 提供VBS启动脚本
2. 高级设置
  - 提供丰富的设置选项
  - 支持自定义Response和Request处理函数
  - 支持Python代码编写以扩展功能
3. 用户体验
  - 现代化的UI设计
  - 窗口拖拽功能
  - 模式开关控制功能启用/禁用
